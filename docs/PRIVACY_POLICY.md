# Startpage 隐私政策

最后更新：2026-06-20

本隐私政策说明 Startpage 网页应用在提供功能时可能处理的数据类型和用途。

## 可能处理的数据

- 账号信息：邮箱、密码哈希、验证码状态、登录会话和推荐码关系。
- 页面数据：快捷链接、便签、订阅源、热榜偏好、主题和搜索偏好等。
- 运行数据：接口请求时间、错误状态、必要的部署平台日志。
- 第三方数据：天气、热榜、壁纸、邮件验证码等功能会调用对应第三方服务。

## 数据用途

这些数据用于登录注册、同步用户配置、展示首页内容、发送验证码、限制滥用和排查故障。Startpage 不以出售用户数据为目的设计。

## 本地与云端

部分设置会存储在浏览器本地；账号、推荐码和需要跨设备使用的数据会通过 Cloudflare Worker 与 D1 等云端服务处理。具体保存范围会随着功能调整而变化。

## 用户注意事项

请不要在便签、快捷链接或订阅配置中保存密码、密钥、支付信息等敏感内容。如果你清理浏览器数据，本地保存的设置可能会丢失。

## 第三方服务

项目可能使用 Resend、Cloudflare、Vercel、天气接口、热榜接口、壁纸接口等服务。第三方服务会按照各自政策处理请求数据。

## 变更

隐私政策可能随功能和部署架构调整而更新。重要变化会通过仓库文档或提交记录说明。
